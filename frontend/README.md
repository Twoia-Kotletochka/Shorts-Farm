# Shorts Farm — веб-панель (frontend)

SPA для управления «фермой» вертикальных шортсов. Vite + React + TypeScript + TailwindCSS.
UI и все тексты — на русском. Тёмная тема. Работает на моках (MSW) до готовности роутов бэкенда,
затем переключается на живой API одним флагом.

## Запуск (в составе монорепо, на сервере)

Через общий `docker-compose.yml` из корня репозитория:

```bash
cd /srv/Shorts-Farm
docker compose up -d --build frontend
# панель: http://192.168.0.92:5173
```

Сервис `frontend` монтирует `./frontend` как volume и поднимает dev-сервер Vite с HMR.
`node`/`npm` на хосте не нужны — всё внутри контейнера `node:20-alpine`.

## Режимы данных

Переключается переменной окружения (в `.env` корня или в `environment` сервиса):

- `VITE_API_MOCK=1` — **моки**: запросы перехватывает MSW в браузере, бэкенд не обязателен.
  Удобно, пока роуты бэкенда (фаза D) не готовы. Значение по умолчанию.
- `VITE_API_MOCK=0` — **живой API**: запросы идут на `/api`, который Vite проксирует на сервис
  `api` (`http://api:8000`). CORS на бэкенде открыт.

## Локальная разработка (опционально, если есть Node ≥18)

```bash
cd frontend
npm install
npm run dev          # http://localhost:5173, моки включены через .env.local (VITE_API_MOCK=1)
npm run typecheck
npm run build
```

## Структура

```
src/
  api/         # http-клиент, типизированные вызовы эндпоинтов, React Query хуки
  app/         # точка входа, провайдеры, layout, роутинг
  components/
    ui/        # базовый кит: Button, Input, Card, Modal, Toast, ...
    layout/    # Sidebar, Topbar, AppShell
  features/    # доменные блоки по вкладкам
  pages/       # страницы-вкладки (Дашборд, Библиотека, Создание, Очередь, Шортсы, Субтитры, Настройки)
  mocks/       # MSW: handlers + фикстуры по контракту
  lib/         # утилиты: cn, форматирование, лейблы, хоткеи
  types/       # api.ts — типы контракта (зеркало Pydantic-схем бэкенда)
```

## Контракт

Источник правды — `prompt/01_общая_спецификация.md` + Pydantic-схемы бэкенда. Типы в `src/types/api.ts`
повторяют имена полей бэкенда (snake_case на проводе). При расхождении — синхронизировать через `01`.
